Welcome to Guarded preprocessors

Simple way to mock up web frontends with HAML/Coffee/SASS/SCSS/Less

Installing

It seems that you already done it

Usage

Just run guard in your working directory and edit your source files.

All of them will be automagically compile in html directory of your project

Preprocessors

Right out the box you can use configured preprocessors

  • HAML – for HTML preprocessing
  • Coffescript – for Javascript preprocessing
  • SASS/SCSS/LESS – for CSS preprocessing
